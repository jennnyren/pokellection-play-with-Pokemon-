# model.Pokemon Collection Manager

## Project Overview
A Java application for managing and exploring model.Pokemon data using the PokeAPI.

## Features
- Fetch model.Pokemon data from PokeAPI
- View model.Pokemon collection
- Search and filter model.Pokemon
- Save/Load collections in JSON and CSV
- Graphical User Interface with Swing

## Prerequisites
- Java 11 or higher
- Maven for dependency management

## Required Dependencies
- Gson (JSON processing)
- JUnit (Testing)
- Java HTTP Client

## How to Run
1. Clone the repository
2. Install dependencies
3. Run `PokemonView.java` to launch the application

## Application Functionality
- Fetch model.Pokemon from API
- Save collections to JSON/CSV
- Load saved collections
- Search model.Pokemon by name
- Filter model.Pokemon by type
- Display model.Pokemon details and images

## Testing
Run the test suite in `test/PokemonApplicationTest.java`
